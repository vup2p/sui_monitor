## SUI Install
Run as root

```shell
sudo -i
```

Run in screen

```shell
screen -S SUI
```
Start install
```shell
apt install wget -y
wget https://raw.githubusercontent.com/vup2p/sui_monitor/main/install_sui_fullnode.sh
bash install_sui_fullnode.sh
```
